# SCmini 
A full stack URL shortener app.

[scmini.herokuapp.com](https://scmini.herokuapp.com/)
# Stack
* React (client)
* Express (server)
* Node.js
* MongoDB (Database)
* Heroku (Deployment)
* Bootstrap (UI)

# Setting up dev environment
## Start server (seperately)

`npm run server`

## Run client side in browser (seperately)

`npm run client`

## Run both at the same time 

`npm run start:dev`
